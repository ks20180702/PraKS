个人一些构想
=============

.. 游戏
..     1. 核心需关于人性
..     2. 需有类型于配件的机制，必须包含‘善良’、‘勇敢’、‘智慧’这三个，且必须无任何加成。可以这样介绍。‘无任何加成，愿你永远善良’。
..     3. 必须包含正反两个故事视图，两种选项剧情可以大体相同，但能看见的npc的行为必定不同。
..     4. 真的希望能有深度，折射人性。
..     5. 特别想增加初始选项：在‘施暴者’和‘受害者’间选择一个，可增加隐含选项‘亦正亦邪’，但此选项必定坏结局(得到一些必定会失去一些，若想两种都得到，那一定要失去另一些更加重要的东西)。
..     6. 希望能有一个坏女人引导，类似白夜行。极美但极危险。
..     7. 能增加唇语部分感觉巨有深意，且玩家只是操纵者，本体也有思想，且剧情中会出现玩家听不见的对话(本体和npc交流)，可通过唇语了解隐藏思想。
